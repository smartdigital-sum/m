# Screenshots

Drop PNG (or JPG) files into this folder using the exact filenames below. The main [README.md](../../README.md) already references them, so once the files exist the images render automatically on GitHub.

## Expected filenames

| Filename | What to capture |
|---|---|
| `home.png` | Homepage hero — navbar, 3D background, headline, CTA buttons |
| `services.png` | AI Powered Services section — the horizontally scrolling cards |
| `examcraft.png` | ExamCraft Pro app — wizard step or a generated question paper |
| `bizwrite.png` | BizWrite app — form + generated profile-kit cards |
| `resume.png` | Resume Builder — resume preview with formatting |
| `games.png` | Games & Projects tab — grid of the 10 games |

## Capture tips

- **Size:** ~1280×800 or 1440×900 looks best on the GitHub grid.
- **Format:** PNG preferred (crisp UI). JPG is fine for photo-heavy shots.
- **Weight:** keep each file under ~500 KB — use [tinypng.com](https://tinypng.com) or `pngquant` to compress.
- **Theme:** mix dark and light screenshots to show the theme toggle exists.
- **Content:** use realistic demo content, not Lorem Ipsum.
